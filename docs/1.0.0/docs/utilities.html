Utilites #

DezUI provides some basic utility classes to speed up development

  • text-color-primary | black | white - Text color.
  • bg-color-primary | black | white - Background color.
  • dm-serif - DM Serif Display font family.
  • raleway - Raleway font family.
  • source-code-pro - Source Code Pro font family.
  • text-[Color Themes] - Color of text
  • bg-[Color Themes] - Background color
  • align-items-[All property values]
  • text-align-[All property values]
  • justify-content-[All property values]
  • dotted - for dotted style <hr>
  • hero - Give padding: 50px 0;
  • hero hero-full - Give padding: 50px 0px; and height: 100vh;
  • width-[breakpoint properties] | full
