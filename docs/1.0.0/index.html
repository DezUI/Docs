DezUI

Lightweight, Minimalist & Elegant Design

Getting Started
Summary

DezUI is a lightweight, minimalistic & elegant css framework for starting projects with a minimalist and elegant design and We focus on using CSS styles and 100% no use of JavaScript.


Lightweight

Light as a feather & size only less than ~4KB zipped


Elegant Design

Reflects someone who wants to look professional and clean.


Easy to Use

Quick to start with zero compiling or installing necessary.

Installation

There are 3 ways to get started with the DezUI framework in your project. You can install manually or from the CDN.


Install manually

Download the zip and copy the code below. You can choose one of the following, between normal files or compressed files

  1. <link rel="stylesheet" href="/css/core.css">
  2. <link rel="stylesheet" href="/css/core.min.css">

Clone the repo
git clone https://github.com/dezui/dezui.git

Install package
npm i install

Install from CDN

This CDN is very convenient and saves local storage.
NB: In the use of local projects requires an internet network connection!

  1. <!-- Via RawGit -->
  2. <link rel="stylesheet" href="https://ghcdn.rawgit.org/dezui/DezUI/stable/css/core.css">

Buttons

DezUI styles buttons with two main styles, outlined and filled. These styles can easily be overwritten, and are there to provide a base for buttons. The default style is outlined, if you want to make it filled add a class btn-fill. And by default the text will be changed to Uppercase.




This button is available by default, available with:
<button>button</button>
<input type="button | submit">
<a class="btn"></a>
<a class="btn btn-fill"></a>

  2. <input type="button" value="Input Button">

Utilites

DezUI provides some basic utility classes to speed up development

  • text-color-primary | black | white - Text color.
  • bg-color-primary | black | white - Background color.
  • dm-serif - DM Serif Display font family.
  • raleway - Raleway font family.
  • source-code-pro - Source Code Pro font family.
  • align-items-[All property values]
  • text-align-[All property values]
  • justify-content-[All property values]
  • dotted - for dotted style <hr>
  • hero - Give padding: 50px 0;
  • hero hero-full - Give padding: 50px 0px; and height: 100vh;
  • width-[breakpoint properties] | full
