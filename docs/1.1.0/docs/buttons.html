Buttons #

DezUI styles buttons with two main styles, outlined and filled. These styles can easily be overwritten, and are there to provide a base for buttons. The default style is outlined, if you want to make it filled add a class btn-fill. And by default the text will be changed to Uppercase.




This button is available by default, available with:
<button>button</button>
<input type="button | submit">
<a class="btn"></a>
<a class="btn btn-fill"></a>

  1. <button>Button</button>
  2. <input type="button" value="Input Button">
  3. <input type="submit" placeholder="Input Submit">
  4. <a href="#" class="btn">Link</a>
